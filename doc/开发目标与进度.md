### 核心目标
```
一句话描述项目目标：
搭建一个用于 DND / 西征世界的网页化数据管理中心

目标用户：
管理员、普通玩家

成功标准：
- 玩家可用指定角色完成公共商店购买
- 私人物品可在玩家交易区自动成交
- 管理员可追溯关键数值和商店变更
```

## MVP 范围

### 必须做（当前版本）
- [x] 账号登录与管理员创建用户
- [x] 角色、背包、金币/声望基础管理
- [x] 荣誉值管理
- [x] 公共商店购买与半价回收
- [x] 玩家交易自动成交
- [x] 审计日志与后台补齐

### 可延后（下一版本）
- 炼金、种植、锻造小游戏：当前不影响首版闭环
- 自动化规则：经济系统仍可能调整

### 不做（明确排除）
- 玩家自助注册：首版统一由管理员创建
- 议价、拍卖：会明显增加交易复杂度
