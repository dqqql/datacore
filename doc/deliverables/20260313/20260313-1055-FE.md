## 2.1 结论区
- 字体风格统一：已根据最新确认，将全角 `body` 的中文字体由 `Noto Sans SC` 统一修改为了衬线体 `Noto Serif SC`，使得界面与奇幻/跑团背景主题更和谐。
- 文本排布修复：
  - 针对所有操作按钮（`.btn-primary`, `.btn-secondary`, `.btn-danger`）和各类 badge 强加了 `white-space: nowrap;` 属性，解决按钮文本（特别是仅包含1-2字的短标签）在受限空间下错误的上下换行排列的问题。
  - 对于 `/dashboard`（控制台）中由于 Flex 折行可能的空间挤压，“查看角色详情”按钮加入了 `shrink-0`。
  - 对于 `/characters`（角色卡册）表格操作列，修改 Flex 布局容器的折行属性，采用 `flex-nowrap` 防护。
  - 对于 `app-shell`（包含“使用原则”小卡片位置）补充了不换行属性配置。
- 版本控制情况：整体修复已完成验证，并通过 `git commit -m "fix(ui): global serif font and align buttons"` 提交到当前本地分支中。
